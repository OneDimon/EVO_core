## Правила YMS-MMM

### Критические ошибки (блокировка + логирование)
- Удаление или перезапись хронологических данных
- Подмена версий API (Gemini 1.5 вместо актуальной и т.п.)
- Потеря специфических флагов и параметров при «упрощении»
- Артефакты генерации символов (буквы «о», «н», «г», «ч» в медиа-модулях)

### Структуры данных

```python
class VerifyRequest(BaseModel):
    output: str           # вывод флагмана
    original_tz: str      # исходное ТЗ пользователя
    cartridge: str        # картридж, который был выдан флагману
    session_id: str

class VerificationResult(BaseModel):
    passed: bool
    score: float          # 0.0–1.0
    failures: list[str]   # список конкретных нарушений
    session_id: str
```
